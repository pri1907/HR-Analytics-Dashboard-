# HR-Analytics-Dashboard-
## Project Overview
This project is an interactive Power BI dashboard developed to analyze employee attrition, workforce performance, and key HR metrics. It provides actionable insights through interactive reports, KPIs, and visualizations to support data-driven HR decisions.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Report Pages

### 1. HR Analytics Dashboard
- Employee Count
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company
- Attrition by Education
- Attrition by Age
- Attrition by Salary
- Attrition by Job Role
- Attrition by Years at Company
- Department-wise Analysis

## Key Features
- Interactive KPI Dashboard
- Dynamic Filters and Slicers
- Employee Attrition Analysis
- Department-wise Performance Analysis
- Age and Salary Insights
- Job Role Analysis
